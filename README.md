# td
A hierarchical todo list program, written using ncurses and golang.
It saves the todo list in JSON format in `~/.td`.

## Bindings
- `j/k`: Move up/down
- `h/l`: Move outside/inside
- `o`: Create new todo
- `dd`: Delete todo
- `e`: Edit todo title
- `J/K`: Change completeness
- `H/L`: Change weight
- `r`: Redraw
- `ESC/Ctrl+C`: Exit
